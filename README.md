# A task scheduler modal operator implemented for blender.

### Features:
- Launch functions in a direct blocking manner, or in a background multiprocess.
- Support multiple background tasks launch in parrallel.
- Analyse function dependencies & schedule their job.
- Tons of hooks and callbacks to easily adapt to your system with minimal edits.
- Make multiprocessing module work in blender bpy.
